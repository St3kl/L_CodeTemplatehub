# TemplateHub - Free Premium Website Templates

🚀 TemplateHub is a modern landing page designed to showcase and distribute high-quality, responsive website templates for developers, designers, and digital creators.

This project acts as a central hub where users can:
- Preview templates live
- Explore different styles and layouts
- Download templates easily

---

## 🌟 Features

- Clean and modern UI design
- Fully responsive layout
- Template showcase grid
- Direct links to live demos
- Integrated download flow (Gumroad)
- Easy to customize and expand

---

## 🎯 Purpose

The goal of TemplateHub is to provide a simple and effective platform to:
- Present multiple templates in one place
- Build trust with users through live previews
- Drive traffic and downloads
- Serve as a foundation for a scalable template business

---

- **NovaPortfolio** – Clean and simple portfolio template
